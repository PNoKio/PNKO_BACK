# ⭐ Service Introduce
### **시각장애인을 포함한 디지털 취약계층이 겪고 있는 키오스크 사용의 어려움을 해소하기 위해 NFC 태그와 음성 인식을 활용해 주문 정보를 생성 서비스 플랫폼**

1. 매장 내의 부착된 NFC 스티커 태그 시 애플리케이션 실행 및 서버에 매장 정보 및 메뉴 정보 요청
2. 응답 받은 정보를 토대로 음성 인식을 활용한 주문 프로세스 이용하여 주문 정보 생성
3. 주문 생성 후 키오스크의 NFC 리더기에 휴대폰 태그 시 주문 완료
